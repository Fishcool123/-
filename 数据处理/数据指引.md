# 数据指引 — 分析图表与CSV文件索引

**目录结构**：

```
数据处理/
├── 数据分析报告.md          ← 完整分析报告
├── 数据指引.md              ← 本文档
├── 图表_路由网络/           ← 3张图表
```

## 一、图表清单

### 1.1 路由网络结构分析（`图表_路由网络/`，3张）

| 文件名 | 内容说明 |
|--------|----------|
| `图1_路由长度分布.png` | 路由跳数（2~6节点）的分布柱状图；左侧显示跳数分布，右侧显示各跳数对应的节点数 |
| `图2_网络拓扑特征.png` | 4合1图表：Top30繁忙中心排名、分拣中心出现频次分布直方图、Top30出度（始发）排名、Top30入度（目的）排名 |
| `图3_路由模式分析.png` | 4合1图表：Top20第一中转站（路由第二节点）排名、Top20第二中转站排名、路由类型构成饼图（直达/中转）、OD对各路由数分布 |

**用途**：理解网络层级结构、识别核心枢纽、评估路由多样性

### 1.2 货量时间序列分析（`图表_货量时间序列/`，9张）

| 文件名 | 内容说明 |
|--------|----------|
| `E1_货量分布直方图.png` | 货量（对数坐标）的整体分布直方图，展示极度右偏特征（偏度13.35） |
| `E2_帕累托分布.png` | 洛伦兹曲线/Lorenz图：横轴为流向累积百分比，纵轴为货量累积百分比。前20%流向贡献82%货量 |
| `E3_每日总货量趋势.png` | 161天全网每日总货量折线图，标注春节V型波动（2月尖底）和趋势线 |
| `E4_星期效应.png` | 周一~周日每日总货量的箱线图，展示周内波动极小（p=0.93，无显著差异） |
| `E5_月度趋势.png` | 各月总货量/日均货量柱状图，标注环比变化。2月-19.4%，3月+24.2% |
| `E6_Top5流向时间序列.png` | Top-5高货量OD对各自161天每日货量折线图（5条子图），观察高货量流向的季节模式 |
| `E7_Top10流向星期效应.png` | Top-10高货量流向的星期箱线图矩阵（2行5列），验证周内模式在各流向间的一致性 |
| `E8_稀疏性分析.png` | 流向活跃天数分布直方图 + 货量热力图样本（矩阵抽样），展示84%流向几乎全勤 |
| `E9_分拣中心货量排名.png` | Top-20始发站和Top-20目的站的日均货量排名柱状图 |

**用途**：为问题1（货量预测）提供建模依据——确定周期特征（无星期效应）、趋势特征（平稳）、季节性（仅春节）、分布特征（帕累托）

## 三、Python脚本（`Python脚本/`，6个）

| 文件名 | 功能 | 依赖 |
|--------|------|------|
| `explore_data.py` | 初始数据探索，读取附件表1-3的基本信息 | pandas, matplotlib |
| `network_analysis.py` | 路由网络分析：路由长度分布、中心频次、层级分类 | pandas, matplotlib, networkx, numpy |
| `deep_network_analysis.py` | 深度网络分析：图论特征、社区检测、层级结构 | pandas, matplotlib, networkx, numpy, scipy |
| `export_classification.py` | 分拣中心评级分类导出：综合评分计算、五级分类、CSV导出 | pandas, numpy |
| `eda_analysis.py` | 附件表3&4探索性分析：成本产能分布、设备采购盈亏平衡 | pandas, matplotlib, numpy, openpyxl |
| `read_data.py` | 简单的数据读取测试脚本 | pandas, openpyxl |

## 四、快速导航

**想看整体结论？** → `数据分析报告.md`

**想看核心枢纽是谁？** → 图表_路由网络/ 或 CSV数据/分拣中心评级分类表.csv

**想看货量预测怎么做？** → 图表_货量时间序列/（特别是E3趋势、E4星期效应、E5月度趋势）

**想算设备投资划算不？** → 图表_分拣中心成本产能/（特别是图6设备对比、图7回收期）

**想复现分析？** → Python脚本/（按 explore_data → network_analysis → export_classification → eda_analysis 顺序运行）
